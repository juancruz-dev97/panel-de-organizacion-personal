# Panel de Organización Personal

Este proyecto fue desarrollado como una herramienta para estudiantes y personas que buscan organizar mejor su rutina diaria.  
Incluye funcionalidades como gestión de tareas, metas, hábitos y un temporizador Pomodoro para mejorar la productividad.  
La idea principal es ayudar a mantener un equilibrio entre estudio, entrenamiento y actividades personales.

## Características
- Panel central con tareas y metas.
- Temporizador Pomodoro integrado.
- Sección para hábitos diarios.
- Diseño adaptable y fácil de usar.
